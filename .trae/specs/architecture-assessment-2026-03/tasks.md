# MDLooker 网站架构优化任务清单

本任务清单基于架构评估报告，涵盖 4 个新页面开发、功能整合优化、数据采集三大部分。

- [ ] **任务 1.1**: 创建监管机构数据库表结构
  - [ ] 编写数据库迁移脚本 `regulatory_agencies.sql`

  - [ ] 创建索引和约束
  
- [ ] **任务 1.2**: 开发监管机构信息展示页面
  - [ ] 创建页面 `app/[locale]/regulators/page.tsx`
  - [ ] 实现监管机构列表展示 (按国家分组)
  - [ ] 实现监管机构详情页 `app/[locale]/regulators/[id]/page.tsx`
  - [ ] 添加搜索和筛选功能 (按国家、机构类型)
  
- [ ] **任务 1.3**: 监管机构数据采集
  - [ ] 编写数据采集脚本 `scripts/scrapers/regulatory_agencies_collector.py`
  - [ ] 采集 15+ 个主要国家监管机构信息
