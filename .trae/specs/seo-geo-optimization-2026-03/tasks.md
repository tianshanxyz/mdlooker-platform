# MDLooker SEO 与 GEO 优化任务清单

## Phase 1: 基础 SEO 优化 (第 1-2 周)

### 任务 1: 页面速度优化
- [ ] **任务 1.1**: Core Web Vitals 分析与基准确认
  - [ ] 使用 PageSpeed Insights 测试所有关键页面
  - [ ] 记录当前 LCP、FID、CLS 指标
  - [ ] 识别性能瓶颈
  
- [ ] **任务 1.2**: 图片优化
  - [ ] 实施图片懒加载 (loading="lazy")
  - [ ] 转换图片为 WebP/AVIF 格式
  - [ ] 为所有图片添加 width/height 属性
  - [ ] 实现响应式图片 (srcset)
  
- [ ] **任务 1.3**: 字体优化
  - [ ] 预加载关键字体
  - [ ] 使用 font-display: swap
  - [ ] 限制字体文件数量
  
- [ ] **任务 1.4**: JavaScript 优化
  - [ ] 实施代码分割
  - [ ] 移除未使用代码
  - [ ] 延迟加载非关键 JS
  - [ ] 优化第三方脚本加载

**验收标准**:
- LCP < 2.5 秒
- FID < 100 毫秒
- CLS < 0.1
- PageSpeed 分数 > 90
