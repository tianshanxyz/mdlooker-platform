## 📱 AI助手设计方案

### 位置与交互
- **位置**: 网站右下角浮动按钮
- **初始状态**: 圆形AI图标，带微动画吸引注意
- **点击行为**: 展开对话框，覆盖在页面内容上方
- **关闭方式**: 点击X按钮或点击对话框外区域

### UI设计规范
```
- 按钮尺寸: 56px × 56px
- 按钮颜色: 品牌主色 #339999
- 对话框宽度: 380px（桌面）/ 100%（移动端）
- 对话框高度: 520px
- 圆角: 16px
- 阴影: 0 8px 32px rgba(0,0,0,0.15)
- 动画: 展开/收起使用scale+opacity过渡，时长300ms
```

### 功能层级
| 层级 | 功能 | 用户类型 |
|-----|------|---------|
| 基础 | 合规问答、法规解读、基础咨询 | 所有用户 |
| 高级 | 个性化建议、市场分析报告、智能提醒 | 注册用户+ |
| 专家 | 定制化合规方案、专属顾问对接 | VIP用户 |

## 🔧 技术实现要点

### AI助手技术栈
- **API**: 百度文心一言 ERNIE-Speed（免费额度）
- **前端**: React组件 + CSS动画
- **知识库**: Supabase向量数据库
- **RAG**: LangChain + 向量检索

### 搜索优化技术
- **搜索引擎**: Algolia / Elasticsearch
- **自动补全**: Trie树 + 热门搜索缓存
- **搜索结果**: 多维度排序算法

### 数据库扩展
- **新加坡HSA**: 网络爬虫 + 官方API
- **日本PMDA**: PMDA英文数据库
- **中东SFDA**: 各国官方数据源整合
