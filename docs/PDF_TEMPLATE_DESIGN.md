# MDLooker 网站 PDF 下载模板设计方案

## 设计概述

为所有从 MDLooker 网站下载的文件创建统一的视觉识别系统，确保专业性和品牌一致性。

---

## 1. 模板规格

### 页面设置
- **页面尺寸**: A4 (210mm × 297mm)
- **方向**: 纵向
- **边距**: 上下 20mm，左右 15mm
- **主色调**: #339999 (青绿色)
- **辅助色**: #2a7a7a (深青绿)
- **背景色**: #FFFFFF (白色)

## 4. 视觉元素规范

### 4.1 颜色系统

|------|----------|------|
| 主色 | #339999 | Logo、标题、重点 |
| 深主色 | #2a7a7a | 悬停、强调 |
| 浅主色 | #339999/10% | 背景、表格头 |

### 4.2 字体规范

- **等宽字体**: Consolas / Monaco (用于代码、编号)

使用统一图标增强可读性：
- 📋 文档/报告

- 🔍 产品/搜索
- 🌍 市场/国家
- 🏛️ 监管机构
- 📊 数据/统计
- ⚠️ 警告/注意
- ✅ 完成/确认
- 📅 日期/时间
- 💰 费用/成本

## 6. 技术实现建议

### 6.1 推荐库
- **PDF 生成**: `pdfmake` 或 `@react-pdf/renderer`
- **HTML 转 PDF**: `html2pdf.js` 或 `puppeteer`
- **图表支持**: `chart.js` + `chartjs-to-image`

### 6.2 文件命名规范
```
mdlooker-[report-type]-[target]-[YYYYMMDD]-[random].pdf

示例:
- mdlooker-map-china-20240115-a1b2c3.pdf
- mdlooker-company-medtech-20240115-d4e5f6.pdf
- mdlooker-product-stent-20240115-g7h8i9.pdf
```

### 6.3 二维码内容
```
https://mdlooker.com/verify?report=[report-id]&user=[user-id]
```
